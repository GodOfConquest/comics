About the open source project

Functionality

comics is a web comics aggregator. Out of the box it can crawl and archive about two hundred comics every day. The comics are made available through an easy to use web interface were users can build personalized collections of their favorite comics, and then read them on the site or export them using feeds.

Development

comics is licensed under the GNU Affero General Public License version 3. Development of the generic comics software is coordinated through GitHub.

Feedback to the administrator of this specific comics installation may be submitted through the feedback page.

Technology

comics makes use of the Python programming language, the Django web framework, the jQuery JavaScript library, and the Twitter Bootstrap web design toolkit. All output should adhere to the HTML5, CSS, and Atom web standards.
