About

Daily Comics (or comics for short) is web based aggregator for web comics. It was originally developed as a custom web application hosted at comics.jodal.no.

Tips: Keyboard Navigation

If you got JavaScript enabled in your browser, you may navigate the comics using the keyboard. Press the h, j, k, and l keys to go to the previous page, the comic strip below, the comic strip above, and the next page, respectively. This feature was contributed by Thomas Adamcik.

Development

Daily Comics is maintained by Stein Magnus Jodal, with comments and input from numerous users. The software is freely available under the GPLv2 license. You can get the source from http://code.jodal.no/bzr/comics/ using the version control system Bazaar, or browse the source online. Feature requests, bug reports and patches are warmly received, read and replied to at comics@jodal.no and the feedback page.

Colophon

Daily Comics makes use of the Python programming language, the Django web framework, Yahoo! UI Library for style sheets, and icons from FamFamFam. Google Analytics are used to gather anonymous information on how the site is used, which is helpful when improving the site even further. All output should adhere to the XHTML, CSS, and Atom web standards, and comply with Section 508.
