About

Daily Comics is a web based aggregator for web comics. It was originally developed as a custom web application hosted at comics.jodal.no, but the source code has later been released for others to change and use.

Tips: Keyboard Navigation

If you got JavaScript enabled in your browser, you may navigate the comics using the keyboard. Press the h, j, k, and l keys to go to the previous page, the comic strip below, the comic strip above, and the next page, respectively. This feature was contributed by Thomas Adamcik.

Development

Daily Comics is maintained by Stein Magnus Jodal, with comments and input from numerous users. The software is freely available under the GNU Affero GPL version 3. You can get the source using the version control system git and the command git clone git://github.com/jodal/comics.git, or you can browse the source online.

Feature requests, bug reports and patches are warmly received, read and replied to at comics@jodal.no or through the feedback page.

Colophon

Daily Comics makes use of the Python programming language, the Django web framework, Yahoo! UI Library for style sheets, and icons from FamFamFam. Google Analytics is used to gather anonymous information on how the site is used, which is helpful when trying to improve the site even further.

All output should adhere to the XHTML, CSS, and Atom web standards, and comply with Section 508.
