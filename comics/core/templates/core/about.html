About

comics is a web comics aggregator. Out of the box it can crawl and archive more than a hundred comics every day. The comics are made available through an easy to use web interface were multiple users can create sets of their favorite comics. Alternatively, comics provides feeds of both single comics and sets of comics.

Feedback to the site administrator may be submitted through the feedback page.

Colophon

comics makes use of the Python programming language, the Django web framework, Yahoo! UI Library for style sheets, and icons from FamFamFam. All output should adhere to the XHTML, CSS, and Atom web standards.

Tips: Keyboard Navigation

If you got JavaScript enabled in your browser, you may navigate the comics using the keyboard. Press the h, j, k, and l keys to go to the previous page, the comic release below, the comic release above, and the next page, respectively.
